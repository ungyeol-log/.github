# 📱 운결 (運結)

> 운의 흐름이 맺히는 순간, 당신의 선택은 패턴이 됩니다.

운결은 사주/운세를 단순히 소비하는 것을 넘어,  
시간 속에서 쌓이는 선택과 결과를 연결하여  
개인의 운의 흐름과 패턴을 이해하는 데이터 기반 플랫폼입니다.

---

## 💡 Overview

대부분의 운세 서비스는 “보고 끝나는 콘텐츠”입니다.

운결은 이 문제를 해결하기 위해  
운세를 기록하고, 시간이 지난 뒤  
실제 경험과 연결하여 개인의 흐름을 분석합니다.

> 👉 단순한 운세 앱이 아닌  
> **“운의 흐름을 데이터로 해석하는 서비스”**를 지향합니다.

---

## 🎯 Core Concept

운결은 3가지 흐름으로 구성됩니다.

### 1. Prediction (운의 방향)
- 사주 / 운세 기반 흐름 제시  
- 금전, 연애, 커리어 등 카테고리별 제공  

---

### 2. Reality (실제 선택)
- 사용자의 실제 경험 기록  
- 간단한 선택 (👍 / 😐 / 👎) 및 메모  

---

### 3. Insight (운의 패턴)
- 예측과 실제의 연결  
- 개인 행동 패턴 분석  
- 의사결정 흐름 도출  

---

## 🔁 User Flow

1. 오늘의 운 흐름 확인  
2. 자동 저장 (Prediction)  
3. 일정 시간 후 피드백  
4. 실제 경험 입력 (Reality)  
5. 운의 패턴 분석 제공 (Insight)  

> 👉 반복될수록 더 선명해지는 나의 운의 흐름

---

## 📊 Key Features

- 📅 **운의 흐름 기록**
  - 날짜 및 영역별 데이터 축적  

- 🔍 **피드백 기반 연결**
  - 예측과 실제 경험 연결  

- 📈 **패턴 분석**
  - 운의 흐름 적중률  
  - 행동 및 선택 패턴 분석  

- 🧠 **인사이트 제공**
  - 의사결정 흐름 도출  
  - 반복되는 운의 패턴 해석
