# 📱 Memoa Log

> 당신의 선택과 결과를 기록하고,  
> 그 흐름 속에서 나를 이해합니다.

Memoa Log는 사주/운세를 단순히 소비하는 것을 넘어, 기록(Log)하고, 검증하고, 분석하는 데이터 기반 자기 이해 플랫폼입니다.

---

## 💡 Overview

대부분의 운세 서비스는 “보고 끝나는 콘텐츠”입니다.

Memoa Log는 이 문제를 해결하기 위해  
운세를 데이터로 저장하고, 시간이 지난 뒤  
실제 경험과 비교하여 개인의 패턴을 분석합니다.

👉 단순한 운세 앱이 아닌  
**“개인의 삶을 데이터로 이해하는 서비스”**를 목표로 합니다.

---

## 🎯 Core Concept

Memoa Log는 3가지 데이터 흐름으로 구성됩니다.

### 1. Prediction (예측 데이터)
- 사주 / 운세 기반 정보 제공
- 금전, 연애, 커리어 등 카테고리별 저장

---

### 2. Reality (실제 데이터)
- 사용자의 실제 경험 기록
- 간단한 선택 (👍 / 😐 / 👎) 및 메모 입력

---

### 3. Insight (분석 데이터)
- 예측과 실제 비교
- 개인 행동 패턴 분석
- 의사결정 경향 도출

---

## 🔁 User Flow

1. 오늘의 운세 확인  
2. 자동 저장 (Prediction)  
3. 일정 기간 후 피드백 요청  
4. 실제 경험 입력 (Reality)  
5. 분석 결과 제공 (Insight)  

👉 반복될수록 더 정확한 개인 분석 가능

---

## 📊 Key Features

- 📅 운세 기록 (Log)
  - 날짜 및 카테고리별 데이터 축적

- 🔍 피드백 기반 검증
  - 과거 운세에 대한 실제 결과 기록

- 📈 패턴 분석
  - 운세 적중률
  - 행동 유형 분석

- 🧠 인사이트 제공
  - 개인의 의사결정 패턴 도출
  - 반복되는 행동 흐름 분석
